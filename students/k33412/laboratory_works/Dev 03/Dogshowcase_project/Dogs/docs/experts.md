# Show, delete and replace an expert

Shows info about an expert

**URL** : `/experts/<id>`

**Method** : `GET PUT PATCH`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

**Code** : `200 OK`

**Content** : `{}`
